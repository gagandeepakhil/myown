    This html boiler plate is generally the template for all the html pages and code we write.so we see waht this template actually looks like.

  • DOCTYPE

    • In the very first line we find "!DOCTYPE",that actually tells the browser about the document type we are using here.We use html5 in this document.And for older version like htm4 the declaration is very much longer.
    So to declare the document type of html5,we use "!DOCTYPE html" in between tags.
    And now the code tags html,these tells the browser that this is the beginning of html code we write and these tags also need an closing tag.

  • HEAD AND BODY SECTIONS

    • And this code is mainly divided into two sections,*HEAD* section and *BODY* section.This head section generally shows up the information regarding ur html page and this we use an element names meta.There are many attributes we use in meta.
    Generally,the tittle is enclosed in tittle tags.
    Firstly,we discuss charset attribute.This is used to tell the browser in which our page should be encoded.the general character set we all use is unicode in which almost of all languages are included and emojis too are included.If the character set we use doesnt encode correctly or if we use the wrong character set,then the page becomes so messy.
    And now comes the description attribute.This is used to describe ur website while browsing through the search results that came across.This description show just below our tittle.
    And we use keywords attribute in name ,which defines the keywords for the search engine.
    The author attribute is used to represent the author of the page.
    The attribute http-equiv=refresh is used for refreshing the website for every particualr given times say 60 seconds.
    The attriburte viwport is used for scaling and making the width of page fit to the device that we use.
